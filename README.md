# Hotel Paradise

A static landing page for **Hotel Paradise** showcasing luxury rooms, a booking form, smooth scrolling navigation, and a responsive design.

## Overview

This repository contains a simple hotel website built with HTML, CSS, and JavaScript. It features:

- Hero section with luxury branding
- Room listings with pricing and amenities
- Booking form with validation and local storage support
- Smooth anchor scrolling and mobile menu toggle
- Back-to-top button and responsive layout

## Files

- `index.html` - main landing page structure
- `style.css` - styling and responsive layout
- `script.js` - interactive UI behavior and booking form logic

## Usage

1. Open `index.html` in your browser.
2. Navigate through sections using the navigation links.
3. Use the booking form to submit reservation details.

## Notes

- This project is fully static and does not require a server.
- Booking details are saved in the browser's Local Storage for demo purposes.
